# ⚙️ ทิปตั้งค่า (สำคัญ)

1. **Final URL vs Display path** — Final URL = หน้าจริงที่คลิกแล้วไป (ต้องมีจริง) · Display path = แค่ข้อความประดับ (ไม่ต้องมีหน้าจริง)
2. **จับคู่หน้าให้ตรงธีม** — Ad group Pet Friendly → ส่งไปหน้า `/pet-friendly.html` (Quality Score สูง = CPC ถูกลง)
3. **Pin ตำแหน่ง 1** ด้วยคีย์เวิร์ดหลัก (เช่น "ที่พักใกล้อิมแพ็ค เมืองทอง") แต่อย่า pin เยอะ ปล่อยให้ Google หมุนทดสอบ
4. **Conversion:** ตั้ง `booking_form_submit` เป็น Key Event ใน GA4 แล้ว — ลิงก์ GA4 เข้า Google Ads เพื่อ import เป็น conversion (จะ optimize โฆษณาตามการจองจริงได้)
5. **วัดผลแยกหน้า** — ทุก LP มีฟอร์มติดแท็กที่มา (`Landing Page: Pet Friendly` ฯลฯ) และ GA4 ดู conversion ต่อหน้าได้ → เอางบไปทุ่มกลุ่มที่คุ้มสุด
6. **Negative keywords** ที่ควรมี: หางาน, สมัครงาน, ขาย, มือสอง, ราคาถูกที่สุด (ฟรี), เช่ารายเดือน (ถ้าไม่รับ)
